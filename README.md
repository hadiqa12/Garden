# Garden

Information about gardening in your backyard and attracting pollinators.

This repository is a small Next.js website using the App Router. It includes pages for:
- Pest Management
- Companion Gardening
- Plants

## Requirements
- Node.js 18+ recommended
- npm

## Install and run

```powershell
cd "c:\Gardening Website"
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Project structure
- `app/` - Next.js App Router folder with `layout.js` and the site pages.
- `package.json` - scripts and dependencies.
